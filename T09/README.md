# RA3 – Anàlisi de Vulnerabilitats amb OpenVAS

## Descripció
En aquesta activitat realitzo una anàlisi de vulnerabilitats d’un sistema informàtic utilitzant l’eina OpenVAS. L’objectiu és identificar punts febles de seguretat en una màquina vulnerable i proposar mesures per a mitigar-los.

## Resultats d’aprenentatge
- **RA3**: Aplico mecanismes de seguretat activa segons les necessitats del sistema.
- **CA3.3**: Analitzo vulnerabilitats i proposo mesures de correcció mitjançant actualitzacions i configuracions adequades.

## Entorn de treball
- **Màquina d’anàlisi**: OpenVAS (OVA).
- **Màquina objectiu**: Metasploitable-2 (Linux amb vulnerabilitats conegudes).
- Entorn virtualitzat amb VirtualBox.

## Desenvolupament de l’activitat
- Configuro la màquina vulnerable i la màquina OpenVAS.
- Accedeixo a la interfície web d’OpenVAS.
- Creo el host i el target amb les credencials corresponents.
- Executo una exploració de vulnerabilitats sobre la màquina Metasploitable-2.
- Analitzo els resultats obtinguts.

## Anàlisi de vulnerabilitats
Selecciono **quatre vulnerabilitats** detectades i per a cadascuna documento:
- Descripció i **CVE**
- Nivell de gravetat
- Possibles formes d’explotació
- Mesures de mitigació proposades

## Documentació
Tota la documentació del procés i l’anàlisi de vulnerabilitats es troba en aquesta carpeta del repositori, en format Markdown, degudament estructurada.
